# Natural Language to Query Processing

An AI-powered database query assistant that allows users to interact with relational databases using natural language instead of writing SQL queries manually.

## Overview

Natural Language to Query Processing is a DBMS project developed to simplify database interaction for both technical and non-technical users. The system converts natural language questions into SQL queries, executes them on a PostgreSQL database, and presents the results along with visual insights.

This project demonstrates the practical application of Database Management System concepts such as normalization, relationships, joins, aggregations, filtering, query optimization, and data visualization.

---

## Problem Statement

Traditional database systems require users to understand SQL syntax and database structures. This creates a barrier for users who need information but lack technical expertise.

This project addresses the problem by enabling users to query databases using natural language and automatically generating the corresponding SQL queries.

## System Workflow

1. User enters a natural language query.
2. AI interprets the query.
3. SQL query is generated automatically.
4. Query is executed on PostgreSQL.
5. Data is retrieved from the database.
6. Results are displayed in tabular format.
7. Visual insights are generated through charts and graphs.
